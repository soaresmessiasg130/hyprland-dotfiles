# hyprland-dotfiles
Arch Linux + Hyprland

## Requirements
    
These installation steps will do everything for you.

## Installation

- Install dependencies and packages:

```sh
./1-install-dependencies.sh
```

- Backup(or not) you config, REMOVE old configs and link:

```sh
./2-setup-links.sh
```

- Reboot your system:

```sh
./3-reboot-system.sh
```
